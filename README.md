# C# COMPLETO Programação Orientada a Objetos + Projetos

Bem-vindo ao repositório de tarefas do curso de C#  na Udemy! 😊📚

Este repositório contém as tarefas e projetos realizados durante o curso de C# na Udemy. Sinta-se à vontade para explorar.

## Estrutura do Repositório

- 📁 **Class**: Pasta onde armazeno os códigos e exemplos que o professor executa e demonstra durante as aulas.
- 📁 **Exercicio**: Exercícios práticos propostos no curso para reforçar o aprendizado.
- 📁 **program.cs**: Contém o método Main, que chama os exercícios ou class para execução.

## Sobre

Este repositório acompanha o curso C# COMPLETO Programação Orientada a Objetos + Projetos ministrado pelo professor [Nelio Alves](https://www.udemy.com/user/nelio-alves/), disponível na Udemy.

O curso aborda desde os fundamentos da linguagem C# até conceitos avançados de POO, design patterns e desenvolvimento de projetos para consolidar o conhecimento.

## Tecnologias Utilizadas

- Linguagem C#
- .NET Framework / .NET Core (dependendo do projeto)
- Programação Orientada a Objetos
- Desenvolvimento de projetos práticos

## Como usar este repositório

Clone este repositório em sua máquina:

- git clone https://github.com/seu-usuario/nome-do-repositorio.git
- Abra a pasta no Visual Studio ou sua IDE favorita.
- Explore os exercícios e projetos conforme seu interesse.
- Execute e modifique os códigos para aprofundar seu aprendizado.
